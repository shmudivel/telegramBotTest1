# Telegram OpenAI Bot

This is a Telegram bot integrated with OpenAI's GPT model.

## Setup

1.  Clone the repository.
2.  Create a `.env` file with your OpenAI API key and Telegram Bot Token.
3.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
4.       Run the bot:
    ```bash
    python main.py
    ```
